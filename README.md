# Internal Tool – Copan Consultancy Services

This is an internal tool under development for **Copan Consultancy Services**.

The purpose of this tool is to:

* Help manage basic employee relations
* Track and view employee work statistics
* Provide simple insights to support internal decision‑making

## Status

The project is currently under development. Features and implementation may change as development progresses.

## Usage

This tool is intended for **internal use only** and is not meant for external distribution.
